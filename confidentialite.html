Confidentialité

Cette application fonctionne sans compte et sans backend métier. Les données de planning sont stockées localement dans votre navigateur via localStorage.

Les informations saisies ne sont pas envoyées par l'application vers un serveur distant. Si vous changez de domaine, de navigateur ou d'appareil, exportez votre sauvegarde JSON puis réimportez-la sur la nouvelle adresse.

Le site n'intègre actuellement ni analytics, ni pixel marketing, ni outil de session replay.

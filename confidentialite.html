Confidentialité

Cette application fonctionne sans compte et sans backend métier. Les données de planning sont stockées localement dans votre navigateur via localStorage.

Les informations saisies ne sont pas envoyées par l'application vers un serveur distant. Si vous changez de domaine, de navigateur ou d'appareil, exportez votre sauvegarde JSON puis réimportez-la sur la nouvelle adresse.

Mesure d'audience

Le domaine de production passe par Cloudflare pour la diffusion du site, la sécurité et les statistiques de trafic agrégées. Cloudflare peut traiter des informations techniques liées aux requêtes web, comme l'adresse IP, le navigateur, la page consultée, le pays estimé, le référent et des mesures de performance.

Ces statistiques servent à comprendre le nombre de visites et les pays de consultation du site. Elles ne donnent pas accès aux données de planning stockées dans localStorage.

Le site n'intègre pas de pixel marketing ni d'outil de session replay.
